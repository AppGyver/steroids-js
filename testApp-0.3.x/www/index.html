Hello world!

This file is located in www/index.html and it's also missing a proper layout and CSS. If you have an existing PhoneGap project you can place it in www/. But wait until you see what it means to put your project on Steroids.

Let's do that!

Edit config/application.coffee to enable tabs by uncommenting few preconfigured lines: Steroids.config.tabBar.enabled and Steroids.config.tabBar.tabs and the array that defines the tabs.

Then, do a $ steroids push while $ steroids connect is still running and this will get more interesting.
