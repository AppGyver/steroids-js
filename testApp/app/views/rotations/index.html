    • Legacy setAllowedRotations API

  • Note:these API calls will not auto-rotate the screen.
  • Set allowed rotations to portrait.

    API call:
    steroids.view.setAllowedRotations([0])
  • Set allowed rotations to landscape right.

    API call:
    steroids.view.setAllowedRotations([90])
  • Set allowed rotations to all orientations.

    API call:
    steroids.view.setAllowedRotations([90, -90, 0, 180])
  • Set allowed rotations to horizontal rotations.

    API call:
    steroids.view.setAllowedRotations([90, -90])
  • Set allowed rotations to vertical rotations.

    API call:
    steroids.view.setAllowedRotations([0, 180])

    • Legacy steroids.view.rotateTo API

  • Note: these will not work if the target rotation is not among allowed rotations.
  • Rotate screen to landscape left.

    API call:
    steroids.view.rotateTo(-90)
  • Rotate screen to upside-down portrait.

    API call:
    steroids.view.rotateTo(180)
  • Rotate screen to portrait.

    API call:
    steroids.view.rotateTo(0)
  • Rotates screen to landscape right.

    API call:
    steroids.view.rotateTo(90)
